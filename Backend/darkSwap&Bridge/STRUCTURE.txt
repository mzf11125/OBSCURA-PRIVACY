API ENDPOINTS
=============

Health Check
------------
GET  /api/health                               Server health status

Bridge Operations
-----------------
POST /api/bridge/quote                         Get bridge quote
POST /api/bridge/execute                       Execute bridge transaction
GET  /api/bridge/status/:requestId             Check bridge status
POST /api/bridge/solve-usdc                    Solve optimal USDC amount
POST /api/bridge/poll-status                   Poll bridge status

Silent Swap Operations
----------------------
POST /api/swap/quote                           Get swap quote
POST /api/swap/execute                         Execute silent swap


QUICK COMMANDS
==============

Start Server:
  npm start

Test API:
  node test-api.js

Install Dependencies:
  npm install


CONFIGURATION
=============

Required:
  EVM_PRIVATE_KEY                              Your EVM wallet private key

Optional:
  PORT                                         Server port (default: 3000)
  NODE_ENV                                     Environment (development/production)
  EVM_RPC_URL_ETHEREUM                         Ethereum RPC URL
  EVM_RPC_URL_AVALANCHE                        Avalanche RPC URL
  SOLANA_SECRET_KEY                            Solana keypair (for Solana swaps)
  SOLANA_RPC_URL                               Solana RPC URL
  SILENTSWAP_API_URL                           SilentSwap API URL
  SILENTSWAP_ENVIRONMENT                       mainnet/testnet
  API_KEY                                      API authentication key
